# 夜空
在网页上显示月相和行星、恒星。

网页中天体的形状、位置、角度等和实际情况均无关联。

# 效果

## 基本代码
- 星星位置随机
- 星星大小随机
- 星星闪烁随机

## 更新日志

### v1.0.0

- 星星颜色随机
- 加入月相变化
- 月相角度随机
- 月相随小时规律变化
- 行星出现随机
- 行星位置随机
- 行星、恒星高斯模糊
- 月球、行星角度随机
- 当出现行星时出现寻找提示信息

### v1.0.1

- 移除提示信息
- 鼠标在行星之上时会放大且变清晰
- 缩小了行星显示尺寸
- 星星闪烁动画时长随机
- 修改了星星颜色出现的概率
